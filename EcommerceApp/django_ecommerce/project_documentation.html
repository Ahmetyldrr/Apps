1. Proje Genel Bakış

1.1 Proje Tanımı ve Kapsamı

Bu proje, Django 4.2.7 framework'ü kullanılarak geliştirilmiş modern bir e-ticaret uygulamasıdır. Kullanıcıların ürünleri görüntüleyebileceği, sepete ekleyebileceği ve satın alabileceği tam fonksiyonlu bir online alışveriş platformu sunar. Proje, micro-services mimarisi yerine monolitik yapıda geliştirilmiş olup, küçük ve orta ölçekli e-ticaret işletmeleri için idealdir.

Ana Özellikler ve Yetenekler:

  • Ürün Yönetimi: Kategorili ürün sistemi, dinamik stok takibi, yüksek kaliteli görsel yönetimi
  • Gelişmiş Alışveriş Sepeti: Session tabanlı sepet işlemleri, miktar güncelleme, dinamik fiyat hesaplama
  • Kullanıcı Yönetimi: Django'nun güvenli built-in authentication sistemi entegrasyonu
  • Güçlü Admin Panel: Kolay ürün ve kategori yönetimi, gelişmiş filtreleme seçenekleri
  • Responsive Tasarım: Bootstrap 5.3.2 ile mobile-first yaklaşım
  • Güvenlik: CSRF koruması, SQL injection koruması, XSS önleme
  • Performans: Optimize edilmiş veritabanı sorguları, statik dosya yönetimi

4.1.3 Sepet İşlemleri

add_to_cart View
def add_to_cart(request, product_id):
    product = get_object_or_404(Product, id=product_id)
    cart = request.session.get('cart', {})                    # Session'dan sepet al
    
    cart_item = cart.get(str(product_id), {'quantity': 0})   # Mevcut item kontrol
    cart_item['quantity'] += 1                               # Miktarı artır
    cart[str(product_id)] = cart_item                        # Sepete ekle
    
    request.session['cart'] = cart                           # Session'a kaydet
    return redirect('cart_detail')
Önemli: Sepet verileri session'da JSON formatında saklanır. Product ID'ler string olarak kullanılmalıdır.
cart_detail View
def cart_detail(request):
    cart = request.session.get('cart', {})
    cart_items = []
    total_price = 0
    
    for product_id, item_data in cart.items():
        product = get_object_or_404(Product, id=int(product_id))
        quantity = item_data['quantity']
        price = product.price * quantity
        total_price += price
        
        cart_items.append({
            'product': product,
            'quantity': quantity,
            'price': price
        })
        
    return render(request, 'store/cart_detail.html', {
        'cart_items': cart_items,
        'total_price': total_price
    })

4.1.4 Checkout İşlemi

@transaction.atomic
def checkout(request):
    cart = request.session.get('cart', {})
    if not cart:
        return redirect('cart_detail')

    # Stok kontrolü ve güncelleme
    for product_id, item_data in cart.items():
        product = get_object_or_404(Product, id=int(product_id))
        quantity = item_data['quantity']
        
        if product.stock < quantity:
            return redirect('cart_detail')  # Yetersiz stok
        
        product.stock -= quantity
        product.save()

    # Sepeti temizle
    del request.session['cart']
    
    return render(request, 'store/checkout_success.html')
Transaction.atomic: Stok güncelleme işlemi atomik olarak yapılır. Hata durumunda tüm işlemler geri alınır.
